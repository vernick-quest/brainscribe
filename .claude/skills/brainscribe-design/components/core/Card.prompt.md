Surface container — the basic paper block of every view.

```jsx
<Card>Default paper card</Card>
<Card variant="muted">Sunken cream block</Card>
<Card variant="ink">Dark navy panel</Card>
<Card interactive onClick={open}>Clickable session</Card>
```

Variants: `default`, `flat` (no shadow), `raised`, `muted` (cream), `ink` (navy). Pass `interactive` for hover lift.
