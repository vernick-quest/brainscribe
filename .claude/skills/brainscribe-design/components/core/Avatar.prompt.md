Round avatar for tutors, students, or photos.

```jsx
<Avatar tutor="sage" name="Sage" size="lg" />
<Avatar name="Maya" />
<Avatar src="/photo.jpg" name="Mr. Lee" />
```

Pass `tutor` (`sage`/`zip`/`coach`/`muse`) to color by persona, `color` for a custom background, `src` for a photo, or just `name` for an initial. Sizes: `sm` / `md` / `lg`.
